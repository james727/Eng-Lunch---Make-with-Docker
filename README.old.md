# Linting
First, install shellcheck
```
brew install shellcheck
```
(if on mac)
Then, run the lint:
```
./scripts/lint.sh
```

# Run the app
First, install node from https://nodejs.org/en/download/ (or use NVM). Make sure to download version 10!!

Then, install dependencies:
```
npm install
```
Then, run the app:
```
node app.js
```
